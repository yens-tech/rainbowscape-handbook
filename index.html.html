분수 설계 지식
📐 예산 규모별 분수 크기 기준
예산분수 규모 (폭 기준)비고
30억W 약 40m-
50억W 60~70m-
70억W 70~90m-
100억+W 100~130m100억 이상은 길게 늘리기보다 폭이 두툼해짐
💡 분수 기획 핵심 원칙
  • 음악분수 최소 10억부터 시작
  • 바닥분수 노즐 간격 500mm 기준
  • 멀티미디어 영상·레이저 공사 시 기본 6~7억 추가 필요 (최소 20억대, 안정적 30억대)
  • 투시뷰(입면도, 평면도, 배치도): 규모 정해놓고 작업 필수
  • 물 튀김 고려해서 분수 높이 결정, 반영 이미지·디테일 살리기
  • 호수 분수라도 수위 낮으면 부양식 미사용
  • 분수 색감: 빨/노, 파/흰, 빨/초 또는 보색 대비
  • 해양분수 = 해수 부식 가능성 → 스테인리스 자재 사용 (비용 증가)
  • 폭포·벽천 근처에는 소나무 식재 권장 (프리미엄 식재)
  • 벽천은 상부 수조 필요, 직선 사각형태로만 연출 가능
  • 클린로드 지하철·저수지 없는 경우 → 빗물 저장소 별도 계획 필요

소프트웨어 사용법
Adobe Photoshop
주요 작업 팁
  • 이미지 해상도 높이기: 필터 > 뉴럴필터 > 슈퍼줌
  • 누끼 따기: 채널로 마스크 → 선택&마스크 >> on white 100%, 색상정화 100%
  • 노말맵 만들기: 필터 > 3D > 표준맵 생성
  • 분수 반사: 필터 > 왜곡 > 유리 (없으면 필터갤러리) → texture load
  • 파노라마: Automate > Photomerge > Reposition
  • 야경: Filter > Render > Lighting effect > Omni, 조정레이어 > 색상검색 > nightfromday
  • 화이트밸런스: Filter > Camera Raw Filter
  • 하늘 누끼: 채널 복사 Blue → Ctrl+L 조정 → Dodge 하이라이트 밝히기
  • PPT 표지 고화질 내보내기: 저장 후 100% full 화면캡처 → 포토샵에서 crop to media box → resolution 72 → canvas 조정
Adobe Illustrator
주요 작업 팁
  • 폰트 정렬: 정렬탭 > 글리프 테두리에 정렬 > 포인트 텍스트
  • 대지 정리: 속성패널 > 전체 재정돈 (열 간격 입력)
  • 글씨 테두리: Window > Appearance(Shift+F6) > Add new fill → fx > Path > Offset path → Add new effect > Pathfinder > Add
  • 비균일 도형 균일 여백: 오브젝트 > 패스 > 오프셋패스
  • 패스 도형 안내선: Ctrl+5
  • 흑백 변환: 색 선택 > S 0% 또는 편집 > 색상편집 > 흑백설정
  • 같은 색 부분 선택: 마법봉 툴 클릭 후 선택
  • 도면 삽도: .dwg 파일 그대로 불러와야 깨지지 않음
PowerPoint 보고서
작업 기준 & 팁
  • 슬라이드 크기: W 29.704cm × H 21.004cm (인쇄 시 반드시 확인)
  • 고화질 내보내기: 300dpi (레지스트리 편집기 수정 필요) → JPG 내보내기 후 PDF 변환 → 제본
  • 폰트 TTF로 다운받아 사용, 맑은 고딕 주로 사용
  • 텍스트 윤곽선: 0.75pt 두께
  • 그림 서식 > 도형 옵션 > 텍스트 상자 > 자동 맞춤 안 함 설정
  • 동영상 썸네일: 비디오 형식 > 포스터 프레임 > 현재 프레임
  • 슬라이드 마스터 수정: 보기 > 슬라이드 마스터

Blender 핵심 정리
💡 모델링 시 그룹·이름 정리 필수 / 주변 건물 창문 뚫기는 앞열만
🗂️ 필수 세팅 & 주의사항
  • CAD → Blender 불러올 때 Units: meters 미설정 시 → Blender에서 scale 0.001
  • FBX 내보내기 (재질 포함): Path mode > Copy → 오른쪽 상자모양 버튼 활성화 (Embed textures)
  • 3DS Max로 넘길 때 camera, light 삭제 (필요한 것만)
  • 오브젝트 잘려 보일 때: N > View > Clip end 값 늘리기
  • 카메라 렌더링 잘릴 때: Camera 선택 > Camera properties > Clip end 늘리기
  • BlenderGIS: 3.6 버전에서만 정상 작동 (4.0 이상 오류)
  • 지적도 FBX 불러올 때: 윗면만 남기고 > merge by distance > boundary loop
  • Subdivision Iterations 5 이상 넘기면 렉 발생
  • 지도 이미지 Everything으로 찾아놓고 따로 저장 후 Blender 저장 시 같이 넣기
